# WebCore Systems

Portfolio profesional desarrollado con React + Vite.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Estructura

- `src/components`: componentes reutilizables
- `src/styles`: estilos globales
- `public/images`: placeholders visuales de proyectos

## Notas

- El sitio está hecho con CSS custom, sin Tailwind.
- Incluye navbar, hero, sobre mí, proyectos, skills, servicios, contacto y footer.
- Está preparado para personalizar textos, imágenes y enlaces reales cuando quieras.
